# 🩺 MedSafe AI

**MedSafe AI** is an AI-powered medical safety assistant that helps users analyze medication interactions, understand prescription details, and ask medication safety questions using generative AI.

The system provides an intuitive dashboard where users can check drug compatibility, ask medical safety questions, and receive AI-generated insights from prescription text.

## 🚀 Features

💊 **Drug Interaction Checker**
Analyze potential interactions between two medications and receive AI-generated safety insights.

🤖 **AI Medical Assistant**
Ask medication or safety-related questions and receive informative responses powered by a large language model.

📄 **Prescription Analyzer**
Paste prescription details to identify medicines and receive AI-based safety notes.

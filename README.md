# 🩺 MedSafe AI

MedSafe AI is an **AI-powered medical safety assistant** that helps users analyze drug interactions, understand prescriptions, and ask medication safety questions.

## 🚀 Features

* 💊 **Drug Interaction Checker** – Analyze safety between two medicines
* 🤖 **AI Medical Chat** – Ask medication safety questions
* 📄 **Prescription Analyzer** – Extract insights from prescription text
* 🌙 **Light/Dark Mode UI**

## 🛠 Tech Stack

* React + Vite
* Tailwind CSS
* OpenRouter (LLaMA AI)


## ⚠️ Disclaimer

This project is for **educational purposes only** and should not replace professional medical advice.
